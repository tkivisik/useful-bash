## Monitoring the System

### top
top displays processor activity and active tasks of the system.

Example:  
```bash
$ top
```

## Finding Files

### locate
Example:  
```bash
$ locate "somefile.txt"
$ sudo updatedb
```

### find

Example:  
```bash
# Find all files ending with .txt recursively in a folder called Documents
$ find Documents -iname "*.txt"
```
